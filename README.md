## 👔 주요 기능

(웹 버전과 모바일 버전의 기능 90% 일치하여, 모바일은 모바일이라 차별점을 둔 부분만 설명)

### 메인 페이지

- **Web** : 사이트 전체의 첫인상이라 심플하고 깔끔한 느낌으로 제작

  - Navigation bar → 클릭하면 주황색으로 변경되며 해당 페이지로 이동, 호버하면 오픈되는 Mega menu

  - 중반부의 Image carousel → 자동/수동 모두 기능하도록 구현

  - Slide banner → 자동/수동 모두 기능하도록 구현

  - Footer → 전자상거래 페이지에서 반드시 명시해야하는 내용 포함

- **모바일** :

  - Header, Navigation bar :

    → 대부분 페이지에서 Nav. bar 상단 고정

    → 일부 페이지는 간략화된 header 사용하여 깔끔한 느낌을 유지할 수 있게 제작

  - 하단 Menu bar :

    → 대부분 페이지에서 하단 고정

    → 유저가 언제든 원하는 곳으로 이동할 수 있어 편의성 향상

### ABOUT (Web)

- 텍스트 mask animation과 페이지 하단의 재생 가능한 영상으로 액티브한 이미지 구현

### 맨, 우먼, 세일

- **Web** :

  - 상품 컬러별 사진 → 컬러를 클릭 시 해당 컬러에 맞는 이미지로 미리보기 사진이 변경

  - Category bar → 각 페이지에 맞는 항목들로 구성

  - Filter bar :

    → 좌측에 컬러, 사이즈, 가격, 혜택 등 항목으로 클릭하면 오픈되는 '필터'를 구성하여 유저의 편의성 향상

    → 우측에 신상품순, 판매순, 저가순, 고가순 등으로 '보기 옵션' 정렬 가능

  - SALE 페이지의 Slide banner : 브랜드의 액티브한 이미지를 위한 자동 슬라이드 배너

- **모바일** :

  - '필터' 기능과 '보기 정렬'의 dropdown 기능을 한 공간 안에서 구현

### 상세보기

- **Web** :

  - 사진 확대 → 작은 이미지를 클릭하면 확대되는 기능

  - 사이즈 선택의 dropdown

  - 찜하기 → 빈 하트를 클릭하면 주황색으로 변경

  - 추천상품 banner → 특정 상품에 호버하면 테두리가 주황색으로 변경

  - `장바구니`, `구매하기`, `상품평` 버튼 클릭시 이동 가능

- **모바일** :

  - 제품 상세 사진 드래그 → 모바일 환경에서 유저가 편리하게 넘겨볼 수 있는 기본적인 기능

  - `장바구니`, `구매하기` 버튼 하단 고정

    → 유저는 언제든지 결정할 수 있고, 유저의 결정으로 기업은 이익을 볼 확률이 높아짐

### EVENT

- **Web** :

  - Slide banner → 자동/수동 모두 기능하도록 구현

  - Event card → 다양한 이벤트 카드를 추가하여 유저의 체류시간 극대화

- **모바일** :

  - 출석체크 stamp 기능 추가 → 유저의 방문 횟수와 체류시간을 늘릴 수 있는 기능

  - Event card를 화살표 클릭으로 넘겨보도록 변경

    → 출석체크 이후에도 다양한 이벤트를 구경하며 유저의 체류시간 극대화

### 장바구니

- **Web** :

  - component interaction → 수량에 따라 금액 증가

  - 옵션 변경의 dropdown

### 결제하기

- **Web** :

  - `배송지 변경` 버튼 → 클릭시 배송지 목록 modal 등장

  - 배송메모의 dropdown 기능

- **모바일** :

  - `결제하기` 버튼 하단 고정 → '상세 페이지'와 같은 이유로 적용한 기능

### 회원정보수정

- **Web** :

  - `배송지 관리` 버튼 → 클릭하면 배송지 목록 modal 등장

  - `수정 완료` 버튼 → 클릭하면 안내 modal 등장, 두번째 modal에서 `확인` 클릭하면 'MY PAGE'로 이동
